## このサイトについて

このサイトは、Cory Doctorow 著 *Little Brother*（Tor Books, 2008）の日本語訳を掲載しています。

原著は Creative Commons Attribution-NonCommercial-ShareAlike 3.0 Unported ライセンスのもとで配布されています（[無償ダウンロードページ](https://craphound.com/littlebrother/download/)）。本翻訳は原著のライセンス条件に従って公開するものであり、出版社による公式日本語版ではありません。
