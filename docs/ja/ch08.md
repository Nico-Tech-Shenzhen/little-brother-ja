もし自動テロ検出器を作るなんていう馬鹿げたことをやろうと決めたら、まず学んでおくべき数学のレッスンがある。「偽陽性のパラドックス」と呼ばれるもので、なかなかすごい代物だ。

「スーパーエイズ」という新しい病気があるとしよう。100万人に1人しかかからない。このスーパーエイズの検査を開発して、精度は99パーセントだとする。つまり、99パーセントの確率で正しい結果を出す——感染していれば陽性、健康なら陰性、という具合に。100万人にこの検査を受けさせる。

100万人に1人がスーパーエイズにかかっている。検査を受けた100人に1人は「偽陽性」を出す——実際にはかかっていないのに、検査はかかっていると告げる。「99パーセントの精度」とはそういう意味だ。1パーセントは間違える。

100万人の1パーセントとは?

1,000,000/100 = 10,000

100万人に1人がスーパーエイズにかかっている。無作為に選んだ100万人を検査すれば、本物のスーパーエイズの症例はおそらく1件しか見つからないはずだ。でも検査は、スーパーエイズにかかっている人を*1人*とは判定しない。*1万人*だと判定してしまう。

99パーセント精度の検査が、99.99パーセントの*不正確さ*を発揮してしまうのだ。

これが偽陽性のパラドックスだ。本当に珍しいものを見つけようとするとき、検査の精度は、探しているものの希少さに見合っていなければならない。画面上のたった一つのピクセルを指し示そうとするなら、尖った鉛筆でも十分だ——鉛筆の先はピクセルよりずっと小さい(精密だ)から。でも、画面上のたった一つの*原子*を指し示すには、鉛筆の先では役に立たない。それには、先端が原子一個分かそれ以下の幅しかないポインター——検査——が必要だ。

これが偽陽性のパラドックスであり、これがテロにどう当てはまるかというと——

テロリストは本当に珍しい存在だ。ニューヨークのような人口2000万人の都市でも、テロリストは1人か2人しかいないかもしれない。多く見積もっても10人くらいだろう。10/20,000,000 = 0.00005パーセント。2万分の1パーセントだ。

たしかにこれはかなり珍しい。さて、その街の銀行記録や、通行料パスの記録、公共交通機関の記録、通話記録をふるいにかけて、99パーセントの確率でテロリストを捕まえられるソフトウェアがあるとしよう。

2000万人のプールの中で、精度99パーセントの検査は20万人をテロリストだと判定してしまう。でも実際にテロリストなのは10人だけだ。10人の悪党を捕まえるために、20万人もの無実の人間を引っ張って調べなければならない。

ちなみに、テロ検査の精度は99パーセントに*程遠い*。もっと言えば60パーセントくらい。時には40パーセントということさえある。

つまりこれは、国土安全保障省が自ら盛大な失敗のお膳立てをしていたということだ。あり得ないほど稀な出来事——ある人物がテロリストであること——を、不正確なシステムで見つけ出そうとしていたのだから。

僕たちがこんなにも大きな混乱を引き起こせたのも、当然と言えば当然だった。

ウィンナードッグ氏は明らかに「重要な誰か」だ。高そうな時計をしていて、いつも上等なスーツを着ている。僕は勝手に、金融街で働いているんだろうと思い込んでいた。

今日、彼とすれ違いざまに触れ合った瞬間、革ジャケットのポケットにあらかじめ仕込んでおいたアーフィド・クローナーを作動させた。クローナーは、彼のクレジットカードや車のキー、パスポート、財布の中の百ドル札から番号を吸い上げた。

それと同時に、他の人とすれ違ったときに取得しておいた新しい番号を、その一部に書き込んでもいた。まるでたくさんの車のナンバープレートを入れ替えるようなものだったけど、目に見えず、一瞬で終わる。僕はウィンナードッグ氏に申し訳なさそうな笑みを向けて、階段を下り続けた。三台の車のところで少し立ち止まり、前日に通り過ぎたすべての車から取得した番号で、ファストラックのタグを入れ替えた。

ちょっと攻撃的すぎると思われるかもしれないけど、他の多くのXnet利用者に比べれば、僕はまだ慎重で控えめなほうだった。UCバークレーの化学工学専攻の女子学生二人組は、家庭にある品物から爆発物探知機に反応する無害な物質を作る方法を編み出していた。教授たちのブリーフケースやジャケットにそれをまぶして楽しんだ挙句、隠れて様子をうかがい、その教授たちが講堂や図書館に入ろうとして、あちこちに新設された警備班に飛びかかられるのを見物していた。

他にも、封筒に炭疽菌の検査で陽性反応が出る物質をまぶす方法を突き止めようとする人たちもいたけど、他のみんなは正気の沙汰じゃないと思っていた。幸い、彼らがそれを実現できそうな様子はなかった。

サンフランシスコ総合病院の前を通ると、正面玄関の長い行列を見て、満足げにうなずいた。もちろんそこにも検問所があって、インターンやカフェテリアの従業員などとして働くXnet利用者が十分な数いたおかげで、みんなのバッジがぐちゃぐちゃに入れ替わっていた。セキュリティチェックのせいで全員の勤務時間が一時間余分にかかるようになったと読んだし、病院が何とかしなければ組合はストを構えていた。

数ブロック先では、BARTの行列がさらに長くなっているのを見た。警官たちが列の中を行ったり来たりして人を指さし、脇に呼んで尋問したり、バッグ検査やボディチェックをしたりしていた。この行為で何度も訴えられているようだったけど、それで彼らの手が緩む様子はなかった。

少し早めに学校に着いたので、22丁目通りまで歩いてコーヒーを買いに行くことにした——すると、車を二次検査のために止めている検問所の前を通り過ぎた。

学校も負けず劣らず大混乱だった——金属探知機のところにいる警備員は、生徒の学生証にも探知機をかざして、動きが妙な生徒を尋問のために引っ張り出していた。言うまでもなく、僕たちの動きは全員かなり妙なものだった。言うまでもなく、授業は一時間以上遅れて始まった。

授業はめちゃくちゃだった。誰も集中できていなかったと思う。先生二人が、前日の帰宅にどれだけ時間がかかったかを話し合って、今日は早めにこっそり帰ろうと計画しているのが聞こえた。

笑いをこらえるのに必死だった。偽陽性のパラドックス、またしても炸裂!

案の定、授業は早く終わり、僕は遠回りして帰路についた。ミッション地区をぐるりと回って、混乱の様子を見物した。長い車の列。ブロックを取り囲むように並ぶBART駅の行列。不審な活動があったとして口座を凍結され、お金を出してくれないATMに悪態をつく人々(これが、当座預金口座をファストラックやファストパスに直結させることの危険性というものだ!)。

家に帰ってサンドイッチを作り、Xnetにログインした。いい一日だった。街中の人たちが、自分たちの成功を自慢し合っていた。僕たちはサンフランシスコの街を完全に停止させたのだ。ニュース報道もそれを裏付けていた——DHSが暴走している、テロから僕たちを守るはずだったインチキ「セキュリティ」のせいだ、と報じていた。サンフランシスコ・クロニクル紙のビジネス面は、失われた勤務時間や会議などによるDHSセキュリティの経済コストの試算に、一面まるまる使っていた。クロニクル紙の経済学者によれば、この馬鹿げた状況が一週間続けば、ベイブリッジ爆破よりも街に大きな経済的損害を与えるという。

ムハハハハ。

一番いい部分——父はその夜、遅く帰宅した。とても遅く。*三時間*も遅れて。なぜか?止められて、身体検査をされて、尋問されたからだ。そしてそれが*また*起きた。二回も。

二回も!
