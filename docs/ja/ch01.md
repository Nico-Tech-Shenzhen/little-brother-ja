# 第1章

> [!NOTE]
> このページは、第1章冒頭のパイロット翻訳です。翻訳ルールと登場人物の語り口を検討するための試訳であり、第1章全体の翻訳ではありません。

> **[BakkaPhoenix Books, Toronto, Canada](http://www.bakkaphoenixbooks.com/)**
>
> この章をカナダ・トロントの BakkaPhoenix Books に捧げる。バッカは世界最古のSF専門書店であり、今の私というミュータントを作ってくれた場所だ。初めて足を踏み入れたのは10歳ごろ。おすすめを聞いてみたら、タニヤ・ハフ（そう、あのタニヤ・ハフ――ただし当時はまだ有名作家じゃなかった！）が古本コーナーに連れていって、H・ビーム・パイパーの『リトル・ファジー』を手に押しつけてくれた。あの本は私の人生を永遠に変えた。18歳になるころにはバッカで働いていた――タニヤが専業で書くために店を辞めたあとの後任として――そこで、人はどうやって本を選び、なぜ買うのか、一生ものの教えを学んだ。作家なら誰でも書店で働くべきだと思う（実際、長年にわたって多くの作家がバッカで働いてきた！店の30周年記念には、バッカ出身の作家たちのアンソロジーを刊行した。ミシェル・サガラ（別名ミシェル・ウェスト）、タニヤ・ハフ、ナロ・ホプキンソン、タラ・タランたちが参加し――そして私も！）
>
> BakkaPhoenix Books: http://www.bakkaphoenixbooks.com/ 697 Queen Street West, Toronto ON Canada M6J1E6, +1 416 963 9993

僕はサンフランシスコのミッション地区にあるセザール・チャベス高校の3年生だ。つまり、世界でもっとも監視されている人間の一人ってことだ。名前はマーカス・ヤロー。でも、この物語が始まったころ、僕は w1n5t0n という名前で通っていた。読み方は「ウィンストン」。

「ダブル・ユー・ワン・エヌ・ファイブ・ティー・ゼロ・エヌ」じゃない――今もインターネットを「情報スーパーハイウェイ」と呼ぶような、時代遅れのとんちんかん生徒指導担当者でもないかぎりは。

そんな人物を、僕はじつは一人知っている。名前はフレッド・ベンソン、セザール・チャベスの副校長の一人だ。こいつは人間の姿をした、胸にぽっかり開いた傷口みたいなやつだ。でも、どうせ看守につかれるなら、切れ者より間抜けのほうがましだ。

ある金曜日の朝、校内放送（PA）でそいつが言った。「マーカス・ヤロー」と。校内放送はもともと音質が最悪で、そこにベンソンのいつものモゴモゴしゃべりが加わると、学校の放送というよりひどいブリトーを消化しようともがく人間みたいな音になる。でも、雑音の中から自分の名前を聞き取るのは人間の得意技だ――これは生存本能の一つだ。

僕はバッグをつかんでノートパソコンを四分の三ほど閉じた――ダウンロードが飛んじゃうのは嫌だったから――そして避けようのない事態に備えた。

「ただちに事務局にご出頭ください」

ガルベス先生は呆れたように目をぐるりと回した。僕も同じように返した。学校のお上はいつも僕を目の敵にしていた。学校のファイアウォールを濡れたティッシュ同然に突破したり、歩行認識ソフトウェアを誤魔化したり、追跡用のスニッチチップをぶっ潰したりするからだ。ガルベス先生はいい人で、そういうことは咎めない（とくに、イラクに駐留している兄弟と連絡が取れるように、ウェブメールを手伝っているときは）。

相棒のダリルが、通り過ぎる僕の尻をパシっと叩いた。ダリルとはオムツしてた保育所時代から脱走仲間で、ずっとトラブルに引き込んでは、そこから連れ出してきた。僕は両腕をボクサーみたいに頭上に掲げて社会科の教室を後にし、事務局まで容疑者の見せしめ連行よろしく歩き出した。

途中で携帯が鳴った。これも御法度だ――チャベス高校では携帯は *muy prohibido*（超禁止）<!-- TODO: human review — Spanish phrase reflects Mission District Latin culture; keep verbatim per glossary -->――でも、だからって止まるわけない。僕はトイレに飛び込んで、真ん中の個室に入った（一番奥はいつも最悪に汚い。臭いと気持ち悪さを避けようとして、みんな真っ先に奥へ突進するから。賢いやり方も、衛生面でも、真ん中を選ぶのが正解だ）。携帯を確認した。家のPCからメールが届いていた――ハラジュク・ファン・マッドネス（Harajuku Fun Madness）に新情報が上がってる、という知らせだ。ハラジュク・ファン・マッドネスは、人類が発明した中で史上最高のゲームだ。

ニヤッとした。金曜日を学校で過ごすのはどうせテラ最悪だし、逃げ出す口実ができてラッキーだった。<!-- TODO: human review — "teh suck": deliberate 2000s net-speak misspelling; テラ最悪 = period-appropriate Japanese internet intensifier (2ch-era テラ prefix), provisional -->

残りの道のりをぶらぶら歩き、ベンソンの部屋に到着してドアをくぐりながら気軽に手を振った。

「おや、ダブル・ユー・ワン・エヌ・ファイブ・ティー・ゼロ・エヌじゃないか」と彼は言った。フレドリック・ベンソン――社会保障番号545-03-2343、生年月日1962年8月15日、母親の旧姓はディ・ボーナ、出身地はペタルマ――は僕よりはるかに背が高い。僕は小柄な5フィート8インチ（約173センチ）なのに、向こうは6フィート7インチ（約200センチ）だ。大学バスケの現役時代はとうに過ぎ、胸の筋肉はたるんだオヤジおっぱいに変わっていた。タダでもらったドットコム企業のポロシャツ越しにも、嫌でも目についた。いつも今にもダンクシュートをかましそうな威圧感があって、声を張り上げて劇的な効果を狙うのが好きだ。どちらも繰り返しやっていると効き目が薄れてくる。

「あいにく違います」と僕は言った。「そのR2D2とやら、名前も聞いたことがないですね」

「W1n5t0n」と彼はまたスペルアウトした。じろりと睨みつけて、僕がへこたれるのを待った。もちろん、それは僕が何年も使ってきたハンドルネームだった。メッセージボードへの書き込みに使っていたアイデンティティ――「応用セキュリティ研究（applied security research）」への貢献のためにね。つまり、学校を抜け出したり、携帯の監視トレーサーを無効にしたりすること、だ。でも彼には、これが僕のハンドルだとはわからなかった。知っているのはごく少数の人間だけで、その全員を絶対に信頼している。

「ええと、まったく心当たりがないですね」と僕は言った。そのハンドルで、学校ではけっこう派手なことをやってきた――なかでもスニッチタグ・キラーの開発は特に誇りだった――だから二つのアイデンティティを結びつけられたら、まずかった。学校では誰も僕を「w1n5t0n」とも「ウィンストン」とも呼ばない。仲間だってそうだ。呼ぶならマーカス。それ以外はない。

ベンソンは机の後ろに腰を落ち着け、神経質そうに卒業指輪でデスクマットを叩いた。状況が不利になってきたときの彼の癖だ。ポーカーの世界ではこういうのを「テル」と呼ぶ――相手が内心何を考えているかを教えてくれる仕草のことだ。ベンソンのテルは、僕には完全にお見通しだった。

「マーカス、これがどれだけ深刻か、わかってほしいものだな」

「何の話か説明していただければ、すぐに真剣に受け止めますよ、先生」　権力を振りかざす相手をからかうとき、僕はいつもわざと丁寧な言葉を使う。これが僕のテルだ。
