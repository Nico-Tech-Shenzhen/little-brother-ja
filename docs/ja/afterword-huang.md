# あとがき — Andrew "bunnie" Huang

ハッカーとは探検家であり、デジタルの世界の開拓者だ。既存の慣習を疑い、複雑に入り組んだ問題に魅了されるのは、ハッカーの本性である。複雑なシステムはどんなものであれ、ハッカーにとって格好の遊び場になる――その副産物として、ハッカーはセキュリティに関わる問題に自然と引き寄せられる。社会もまた大きく複雑なシステムであり、ちょっとしたハッキングの対象外というわけではない。その結果、ハッカーはしばしば、既成概念を打ち壊す者、社会不適合者、反抗のために社会規範に逆らう者として、型にはめて見られがちだ。2002年にMITでXboxをハックしたとき、私は反抗のためでも、害を与えるためでもなかった。ただ自然な衝動に従っていただけだ――壊れたiPodを直したり、MITの屋根や地下トンネルを探検したりするのと同じ衝動に。

残念ながら、社会規範に従わないことと、クレジットカードのRFIDタグ（arphid）を読み取る方法や、鍵をピッキングする方法といった「物騒な」知識を持っていることが組み合わさると、ハッカーを恐れる人が出てくる。しかし、ハッカーの動機は大抵、「モノを設計するのが好きだからエンジニアになった」というくらい単純なものだ。「なぜXboxのセキュリティシステムをハックしたのか」と聞かれることがよくある。答えは単純だ。まず、私は自分が買ったものを所有している。もし誰かに、自分のハードウェアで何を実行してよくて何がダメか指図されるとしたら、それは所有しているとは言えない。次に、それがそこにあるからだ。それは、遊びがいのある十分な複雑さを持ったシステムだった。博士課程の研究で夜更かししていた日々の、素晴らしい息抜きにもなった。

私は運が良かった。Xboxをハックした当時、私がMITの大学院生だったという事実が、しかるべき人々の目に、その行為を正当なものと映してくれた。しかし、ハックする権利は学者だけに与えられるべきものではない。私がハッキングを始めたのは、まだ小学生の少年だったころで、手に入るあらゆる電化製品を分解しては、両親を困らせていた。私の愛読書には、模型ロケットや火砲、核兵器、爆発物の製造についての本もあった――学校の図書館で借りた本だ（冷戦が公立学校の蔵書選定に影響を与えていたのだと思う）。自作の花火でもさんざん遊んだし、中西部にあった地元の建設中の家々の工事現場にも自由に入り込んでは歩き回っていた。決して賢明なこととは言えないが、これらは私の成長にとって重要な経験であり、地域社会の寛容さと信頼のおかげで、私は自由な思考の持ち主として育つことができた。

昨今の情勢は、これからハッカーを目指す人々にとって、決して優しいものではない。『リトル・ブラザー』は、今の私たちがいる場所から、新しく異質な考えへの社会的寛容さがすっかり失われた世界まで、どうやってたどり着いてしまうのかを描いている。最近起きたある出来事は、私たちが『リトル・ブラザー』の世界との境界線をどれほど際どいところまで越えかけているかを、まさに浮き彫りにした。私は幸運にも、2006年11月に『リトル・ブラザー』の初期草稿を読む機会を得ていた。それから2ヶ月後の2007年1月末、ボストン警察が爆発物と疑われる装置を発見し、街を1日封鎖する事態が起きた。これらの装置は、実はカートゥーン・ネットワークの番組宣伝用に、LEDを点滅させただけの回路基板にすぎなかった。この都市型グラフィティを設置したアーティストたちはテロ容疑者として連行され、最終的に重罪で起訴された。カートゥーン・ネットワークの制作側は200万ドルの和解金を支払う羽目になり、同ネットワークの責任者はこの騒動の責任を取って辞任した。

テロリストはもう勝ってしまったのだろうか？ 私たちは恐怖に屈するあまり、アーティストや趣味人、ハッカー、既成概念を壊す者たち――あるいは、ハラジュク・ファン・マッドネス（Harajuku Fun Madness）に興じているだけの、何の変哲もない子どもたちのグループでさえ――こうもたやすくテロリスト扱いされるようになってしまったのだろうか？

この機能不全には名前がある――自己免疫疾患だ。生体の防御システムが暴走しすぎて自分自身を認識できなくなり、自らの細胞を攻撃してしまう病気である。最終的に、その生体は自壊する。今まさにアメリカは、自らの自由に対してアナフィラキシーショックを起こす寸前にあり、私たちはこれに対する予防接種を自分自身に施す必要がある。テクノロジーは、この被害妄想の治療薬にはならない。むしろ被害妄想を増幅させる可能性さえある――私たちを、自分たちが作った装置の囚人に変えてしまうのだ。何百万人もの人々に、上着を脱がせ、毎日金属探知機を裸足でくぐらせることを強制するのも、解決策にはならない。それはただ、恐れる理由があるのだということを人々に毎日思い出させるだけであり、実際には、本気の敵に対してはお粗末な障壁にしかならない。

真実を言えば、私たちは誰か他の人に、自分を自由な気分にしてもらうことなど当てにできない。そして、被害妄想によって私たちの自由が失われるその日に、M1k3yがやってきて救ってくれることもない。なぜなら、M1k3yはあなたの中に、そして私の中にいるからだ――『リトル・ブラザー』が思い出させてくれるのは、未来がどれほど予測不能であろうと、私たちはセキュリティシステムや暗号、尋問や抜き打ち検査によって自由を勝ち取るのではない、ということだ。私たちが自由を勝ち取るのは、どれほど大きな脅威が地平線の向こうに迫っていようと、毎日を自由に生き、自由な社会として行動する勇気と確信を持つことによってなのだ。

M1k3yのようになろう。ドアの外に一歩踏み出して、自由であることを恐れずに。

Andrew "bunnie" Huang
Xbox Hacker
